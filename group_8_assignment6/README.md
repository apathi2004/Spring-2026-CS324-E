# Assignment 6: Simulation

Our group decided to make a pendulum simulation that can be used to interact strings, balls with mass, and blocks. In our demonstration we created Newton's Cradle, which demonstrates the conservation of momentum through the transfer of energy.

## Componenets
  To make a clean visual hierarchy we broke the pendulum class into two subclasses: 
  - the chain
  - the wrecking ball
    
  Additionally we added a block tower class that is made up off blocks which can all detect collisions and act accordingly

## How to Run
  To run in JetBrains Rider: 
  1. Clone or pull the directory
  2. Navigate in the JetBrains Rider open menu to the .sln file in the assignment6 project directory
  3. Run the program using the play button or `Ctrl` + `F5`
  4. When the program is running, users can interact with the pendulums by left clicking the wrecking balls and dragging them to the user's desired position, and then releasing their mouse
