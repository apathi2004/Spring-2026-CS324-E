# Assignment 7: Tetris

For this assignment we had to make a game and we chose Tetris.
In this game the user can use the arrow keys to move the piece left and right, turn the piece by pressing up, move the piece down by pressing down, or launch it down by pressing space.

We also added a pause screen that pops up when the user presses `p`

## How to run
1. Clone the repo and cd to the directory
2. Run the .sln file with DotNet and the game should start up immediately

In Rider:

1. Open the .sln file
2. Run `assignment7`
3. Begin playing
4. To quit press `Esc`
